# Data Sources for Module 1: Document Analysis with NotebookLM

This module uses materials related to UC Berkeley's student housing crisis for qualitative analysis. Below are the sources needed for the demo.

## Anchor Paper (Required)

**Title:** Crisis by Design: Student Housing and the Hidden Cost of Higher Education

**Authors:** Jiang-Brittan (2025)

**Source:** UC eScholarship Repository

**URL:** https://escholarship.org/content/qt220157sj/qt220157sj.pdf

**Format:** PDF

**Download Instructions:**
1. Visit https://escholarship.org/content/qt220157sj/qt220157sj.pdf
2. Click "Download PDF" to save the file
3. Save as `Jiang-Brittan_2025_Crisis_by_Design.pdf` in this directory

**Description:** This is the anchor paper that frames the research question about how different stakeholders view Berkeley's student housing crisis. It provides academic context for the policy documents and university statements.

---

## UC Berkeley Official Documents

### 1. UC Berkeley Chancellor's Housing Initiative

**URL:** https://chancellor.berkeley.edu/housing-initiative

**Format:** Web page (save as HTML or PDF)

**Download Instructions:**
1. Visit https://chancellor.berkeley.edu/housing-initiative
2. Save the page as HTML (Ctrl+S or Cmd+S, select "Web Page, Complete") OR
3. Print to PDF (Ctrl+P or Cmd+P, select "Save as PDF")
4. Name the file `Chancellor_Housing_Initiative.pdf` or `.html`

**Description:** Official university statement on housing initiatives. This represents the administration's framing of the housing crisis and proposed solutions.

---

### 2. UC Berkeley Housing Task Force

**URL:** https://housing.berkeley.edu/housing-task-force/

**Format:** Web page (save as HTML or PDF)

**Download Instructions:**
1. Visit https://housing.berkeley.edu/housing-task-force/
2. Save the page as HTML or PDF (see instructions above)
3. Name the file `Housing_Task_Force.pdf` or `.html`
4. If available, also download any linked reports or committee findings

**Description:** Information about the university's housing task force, including committee membership, charge, and recommendations.

## City/Policy Analysis Documents

### 3. Policy Review @ Berkeley Housing Analysis

**URL:** https://www.ocf.berkeley.edu/~prb/crisis-control-housing/

**Format:** Web page or document (varies)

**Download Instructions:**
1. Visit https://www.ocf.berkeley.edu/~prb/crisis-control-housing/
2. Save the page as HTML or PDF
3. Name the file `PRB_Housing_Analysis.pdf` or `.html`
4. Look for downloadable PDFs within the page; save these as well

## Document Organization for NotebookLM

Once downloaded, organize documents as follows:

```
module1_document_analysis/
├── README.md
├── SOURCE.md
├── Jiang-Brittan_2025_Crisis_by_Design.pdf         (anchor paper)
├── Chancellor_Housing_Initiative.pdf                (or .html)
├── Housing_Task_Force.pdf                          (or .html)
└── PRB_Housing_Analysis.pdf                        (or .html)
```

Upload all 4 documents to NotebookLM to create a notebook for the demo.
